    How to customize the DRUIDS Mandalorian theme for your organization's needs.

    Understanding the Theme Structure¶

    Current Theme Files¶

    1. druids-material.css - Main theme overrides
    2. Mandalorian color palette
    3. Typography customizations
    4. Component styling

    5. Security indicators

    6. giscus-druids.css - Comment system theme

    7. Matches main theme colors
    8. Custom comment styling

    9. Reaction buttons

    10. overrides/partials/ - HTML template overrides

    11. comments.html - Giscus integration

    Color Palette Customization¶

    Current Mandalorian Colors¶

    :root {
  /* Core Palette */
  --md-primary-fg-color: #7F8C8D;        /* Beskar steel gray */
  --md-accent-fg-color: #C0392B;         /* Mandalorian red */
  --md-default-bg-color: #0A0E27;        /* Deep space black */
  --md-default-fg-color: #ECF0F1;        /* Clean white text */
  --md-code-fg-color: #00FF41;           /* Terminal green */

  /* Security Tiers */
  --druids-l0-color: #27AE60;            /* Public - Clan green */
  --druids-l1-color: #E67E22;            /* Member - Clan orange */
  --druids-l2-color: #C0392B;            /* Cadre - Clan red */
}

    Creating Your Own Palette¶

    Replace the colors in druids-material.css:

    :root {
  /* Example: Revolutionary Red Theme */
  --md-primary-fg-color: #B71C1C;        /* Deep red */
  --md-accent-fg-color: #FFD600;         /* Revolutionary gold */
  --md-default-bg-color: #1A1A1A;        /* Dark background */
  --md-default-fg-color: #FFFFFF;        /* White text */
  --md-code-fg-color: #FF5252;           /* Light red code */
}

    Typography Customization¶

    Current Fonts¶

    :root {
  --font-heading: 'JetBrains Mono', monospace;
  --font-body: 'Inter', system-ui, sans-serif;
  --font-code: 'JetBrains Mono', 'Fira Code', monospace;
}

    Using Custom Fonts¶

    1. Add Google Fonts to mkdocs.yml:
    theme:
  font:
    text: Roboto
    code: Roboto Mono
    1. Or use custom fonts in CSS:
    @import url('https://fonts.googleapis.com/css2?family=Bebas+Neue&display=swap');

.md-typeset h1 {
  font-family: 'Bebas Neue', sans-serif;
}

    Component Styling¶

    Custom Header¶

    /* Gradient header with organization colors */
.md-header {
  background: linear-gradient(135deg, var(--primary-color), var(--secondary-color));
  box-shadow: 0 0 20px rgba(0, 0, 0, 0.5);
  border-bottom: 3px solid var(--accent-color);
}

    Custom Buttons¶

    /* Themed buttons with hover effects */
.md-button {
  background: var(--primary-color);
  border: 2px solid var(--accent-color);
  transition: all 0.3s ease;
  position: relative;
  overflow: hidden;
}

.md-button::before {
  content: '';
  position: absolute;
  top: 0;
  left: -100%;
  width: 100%;
  height: 100%;
  background: var(--accent-color);
  transition: left 0.5s;
}

.md-button:hover::before {
  left: 0;
}

    Adding Custom Effects¶

    Animated Background¶

    /* Subtle animated background */
body {
  background: #0A0E27;
  background-image: 
    radial-gradient(circle at 20% 50%, rgba(120, 119, 198, 0.3), transparent 40%),
    radial-gradient(circle at 80% 80%, rgba(255, 119, 198, 0.3), transparent 40%);
  background-attachment: fixed;
  animation: drift 20s ease-in-out infinite;
}

@keyframes drift {
  0%, 100% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
}

    Glowing Security Indicators¶

    /* Pulsing security badges */
.security-indicator {
  animation: pulse 2s ease-in-out infinite;
}

@keyframes pulse {
  0%, 100% { 
    box-shadow: 0 0 10px currentColor;
    transform: scale(1);
  }
  50% { 
    box-shadow: 0 0 30px currentColor;
    transform: scale(1.05);
  }
}

    Theme Variations¶

    Light Mode Support¶

    /* Light mode overrides */
[data-md-color-scheme="default"] {
  --md-default-bg-color: #FFFFFF;
  --md-default-fg-color: #1A1A1A;
  --md-code-bg-color: #F5F5F5;
  --md-code-fg-color: #D32F2F;
}

    High Contrast Mode¶

    /* Accessibility: High contrast */
@media (prefers-contrast: high) {
  :root {
    --md-default-bg-color: #000000;
    --md-default-fg-color: #FFFFFF;
    --md-accent-fg-color: #FFFF00;
  }
}

    Creating Theme Variants¶

    Option 1: CSS Variables¶

    Create themes/ directory with variants:

    /* themes/forest-revolution.css */
:root {
  --theme-primary: #228B22;
  --theme-accent: #8B4513;
}

/* themes/urban-uprising.css */
:root {
  --theme-primary: #424242;
  --theme-accent: #FF6F00;
}

    Option 2: Multiple CSS Files¶

    In mkdocs.yml:

    extra_css:
  - stylesheets/druids-material.css
  - stylesheets/themes/summer-campaign.css  # Seasonal theme

    Advanced Customizations¶

    Custom Icons¶

    /* Replace default icons with custom SVGs */
.md-nav__link[href*="security"]::before {
  content: '';
  display: inline-block;
  width: 1em;
  height: 1em;
  background: url('data:image/svg+xml;utf8,<svg>...</svg>');
  margin-right: 0.5em;
}

    Page Transitions¶

    /* Smooth page transitions */
.md-content {
  animation: fadeIn 0.3s ease-in;
}

@keyframes fadeIn {
  from { 
    opacity: 0;
    transform: translateY(20px);
  }
  to { 
    opacity: 1;
    transform: translateY(0);
  }
}

    Custom Scrollbar¶

    /* Themed scrollbar */
::-webkit-scrollbar {
  width: 12px;
}

::-webkit-scrollbar-track {
  background: var(--md-default-bg-color);
  border-left: 1px solid var(--md-accent-fg-color);
}

::-webkit-scrollbar-thumb {
  background: var(--md-primary-fg-color);
  border-radius: 0;
}

    Testing Your Theme¶

    Local Testing¶

    1. Save changes to CSS files
    2. Run mkdocs serve
    3. Hard refresh browser (Ctrl+Shift+R)
    4. Test in different browsers

    Accessibility Testing¶

    /* Test with forced colors */
@media (prefers-color-scheme: dark) { }
@media (prefers-reduced-motion: reduce) { }
@media (prefers-contrast: high) { }

    Performance Considerations¶

    • Keep CSS file size under 100KB
    • Use CSS variables for repeated values
    • Minimize animation complexity
    • Test on slower devices

    Sharing Your Theme¶

    Package as Plugin¶

    Create mkdocs-yourtheme-plugin/:

    mkdocs-yourtheme-plugin/

 mkdocs_yourtheme/
    __init__.py
    theme/
       main.html
       assets/
           stylesheets/
               theme.css

    Documentation¶

    Document your theme choices:

    ## Theme Rationale

- **Primary Color**: Represents our organization's main value
- **Accent Color**: For calls to action
- **Background**: Dark for reduced eye strain
- **Typography**: Accessible and revolutionary

    Common Customization Requests¶

    Remove Animations¶

    /* Disable all animations */
* {
  animation: none !important;
  transition: none !important;
}

    Simplified Design¶

    /* Minimalist theme */
.md-header { background: white; color: black; }
.md-tabs { display: none; }
.md-sidebar { background: #f5f5f5; }

    Print-Friendly¶

    @media print {
  .md-header, .md-tabs, .md-sidebar, .md-footer { 
    display: none !important; 
  }
  .md-content {
    margin: 0 !important;
    padding: 0 !important;
  }
}

    Remember: Good themes enhance content readability while maintaining the revolutionary spirit!
