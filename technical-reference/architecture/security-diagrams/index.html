    Democratic Centralist Git Flow¶

    graph LR
    subgraph "Democratic Phase"
        A[Create Proposal Branch] --> B[Open Pull Request]
        B --> C[3-5 Day Discussion]
        C --> D{Vote}
    end

    subgraph "Centralist Phase"
        D -->|Approved| E[Merge to Main]
        E --> F[Create Implementation Branch]
        F --> G[Execute Decision]
        G --> H[Complete Implementation]
    end

    D -->|Rejected| I[Archive PR]

    How to Use These Diagrams¶

    These diagrams are created using Mermaid, which is supported by:

    • GitHub (renders automatically in markdown files)
    • Obsidian (with Mermaid plugin)
    • MkDocs Material theme (built-in support)

    You can copy and modify these diagrams for your organization's specific needs. The color schemes match the DRUIDS security tier visual indicators for consistency.
