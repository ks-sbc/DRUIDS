    Quick Reference¶

    Color Palettes¶

    Queer Mandalorian (Dark Pride): - Primary: #FF6B35 (Orange) - Accent: #00D9FF (Cyan) - Warning: #FF006E (Pink) - Background: #0A0E27 (Deep Space)

    Security Indicators: - L0 Public: #00FF41 (Green) - L1 Member: #FF6B35 (Orange) - L2 Cadre: #FF006E (Red/Pink)

    Typography Scale¶

    /* Base sizing */
--text-base: 1.1875rem;  /* 19px */
--h1: 3rem;              /* 48px */
--h2: 2.25rem;           /* 36px */
--h3: 1.75rem;           /* 28px */

    Accessibility Minimums¶

    • Normal text: 4.5:1 contrast ratio
    • Large text: 3:1 contrast ratio
    • Focus indicators: 2px solid outline
    • Touch targets: 44px minimum

    Contributing¶

    When contributing to design documentation:

    1. Include visual examples or screenshots
    2. Document accessibility considerations
    3. Provide code examples
    4. Test across browsers
    5. Consider cultural impact
